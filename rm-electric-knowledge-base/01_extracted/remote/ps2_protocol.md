# PS2 遥控手柄协议字节映射

> 本文档仅提炼引脚、时序、帧结构与字节/位映射，不涉及源码实现。
> 出处：`4.PS2遥控手柄使用说明书.pdf` P.3-P.5（轮趣科技 V1.1, 2025-12-10）

---

## 物理接口

PS2 手柄采用 **SPI 全双工同步串行** 通信，主机（MCU）提供时钟。

### 接收器引脚

| 引脚号 | 名称     | 方向    | 说明                |
| --- | ------ | ----- | ----------------- |
| 1   | DI/DAT | 手柄→主机 | 8bit 串行数据，时钟下降沿传送 |
| 2   | DO/CMD | 主机→手柄 | 8bit 串行数据，时钟下降沿传送 |
| 3   | NC     | —     | 空                 |
| 4   | GND    | —     | 电源地               |
| 5   | VDD    | —     | 3~5V              |
| 6   | CS/SEL | 主机→手柄 | 片选，通讯期间保持低电平      |
| 7   | CLK    | 主机→手柄 | 时钟信号              |
| 8   | NC     | —     | 空                 |
| 9   | ACK    | 手柄→主机 | 应答信号（编程时通常不使用）    |

### 时序参数

| 参数 | 值 |
|------|-----|
| 时钟频率 | 250 kHz（4μs），不稳定时可降低频率 |
| 数据采样 | 时钟下降沿 |
| CS 有效期 | 整个通讯周期（9 字节）期间保持低电平 |
| 通讯方式 | 全双工，发送和接收同时完成 |

### STM32 接线示例

| 接收器引脚 | STM32 引脚 |
|-----------|-----------|
| DI/DAT | PA3 |
| DO/CMD | PA2 |
| CS/SEL | PA1 |
| CLK | PA0 |
| ACK | 不接 |

## 通讯流程

1. 主机拉低 CS
2. 主机发送 `0x01`，手柄回复 ID
3. 主机发送 `0x42`（请求数据），手柄回复 `0x5A`（数据来了）
4. 随后手柄连续发送按键数据
5. 9 字节传输完毕，主机拉高 CS

### 手柄 ID

| ID 值 | 模式 |
|-------|------|
| 0x41 | 绿灯模式（数字模式） |
| 0x73 | 红灯模式（模拟模式，摇杆有值） |

---

## 9 字节帧结构

一个通讯周期共 9 个字节，每个字节 8bit，依次按位传送：

| 顺序 | DO（主机→手柄） | DI（手柄→主机） | 说明 |
|------|----------------|----------------|------|
| 0 | 0x01 | idle | 起始命令 |
| 1 | 0x42 | ID（0x41/0x73） | 请求数据 + 手柄回复 ID |
| 2 | idle | 0x5A | "数据来了"标志 |
| 3 | WW | data | 按键数据 1（见位映射） |
| 4 | YY | data | 按键数据 2（见位映射） |
| 5 | idle | data | 右摇杆 X 轴 PSS_RX |
| 6 | idle | data | 右摇杆 Y 轴 PSS_RY |
| 7 | idle | data | 左摇杆 X 轴 PSS_LX |
| 8 | idle | data | 左摇杆 Y 轴 PSS_LY |

## 按键位映射

> 按下 = 0，松开 = 1

### 字节 3（Data[3]）

| Bit | 按键 |
|-----|------|
| 0 | SELECT |
| 1 | L3（左摇杆按下） |
| 2 | R3（右摇杆按下） |
| 3 | START |
| 4 | UP（方向键上） |
| 5 | RIGHT（方向键右） |
| 6 | DOWN（方向键下） |
| 7 | LEFT（方向键左） |

### 字节 4（Data[4]）

| Bit | 按键 |
|-----|------|
| 0 | L2 |
| 1 | R2 |
| 2 | L1 |
| 3 | R1 |
| 4 | △（三角） |
| 5 | ○（圆） |
| 6 | ✕（叉） |
| 7 | □（方框） |

### 字节 5~8（摇杆模拟值）

| 字节 | 摇杆 | 范围 | 方向 |
|------|------|------|------|
| 5 | PSS_RX（右摇杆 X） | 0x00~0xFF | 0x00=左, 0xFF=右 |
| 6 | PSS_RY（右摇杆 Y） | 0x00~0xFF | 0x00=上, 0xFF=下 |
| 7 | PSS_LX（左摇杆 X） | 0x00~0xFF | 0x00=左, 0xFF=右 |
| 8 | PSS_LY（左摇杆 Y） | 0x00~0xFF | 0x00=上, 0xFF=下 |

> 摇杆中值约 0x80（128）。仅在红灯模式（ID=0x73）下有效；绿灯模式下摇杆值为固定 0x80。

---

## MODE 按键说明

| 模式 | MODE 灯状态 | 摇杆行为 |
|------|-----------|---------|
| 模式 1 | 红灯常亮 | 摇杆输出 0~255 线性值（PSS_LX/LY/RX/RY） |
| 模式 2 | 红灯熄灭 | 摇杆值固化为 255，摇杆变化占用 PS2_KEY 变量 |

> 程序设置为不锁存时可用 MODE 按键切换模式；锁存状态时默认为模式 1。

> 出处：`4.PS2遥控手柄使用说明书.pdf` P.7（V1.1 新增）

## 与其他遥控协议对比

| 特性     | PS2          | DBUS               | SBUS           |
| ------ | ------------ | ------------------ | -------------- |
| 接口     | SPI（同步串行）    | UART（异步）           | UART（异步）       |
| 波特率/时钟 | 250 kHz      | 100000 bps         | 100000 bps     |
| 帧长度    | 9 字节         | 18 字节              | 25 字节          |
| 通道数    | 12 按键 + 4 摇杆 | 4 摇杆 + 2 开关 + 鼠标键盘 | 16 通道          |
| 摇杆分辨率  | 8 bit（0~255） | 11 bit（364~1684）   | 11 bit（0~2047） |
| 信号极性   | 正常           | 反相                 | 反相             |
| 通信方式   | 全双工          | 单工（接收）             | 单工（接收）         |
| 需要取反电路 | 否            | 是                  | 是              |
