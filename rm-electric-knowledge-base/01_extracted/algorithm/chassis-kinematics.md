# 底盘运动学解算模块

## 坐标系与正方向

```
         ▲  X+  (前进 Forward)
         │
         │
 Y+ ◄────┘        Y+ = 左方向 (Left)
                  ω+ = 逆时针 (CCW, 从顶部俯视)
```

| 符号 | 含义 | 正方向 |
|------|------|--------|
| `vx` | 底盘 X 轴线速度 (m/s) | 前进 |
| `vy` | 底盘 Y 轴线速度 (m/s) | 左移 |
| `vw` / `ω` | 底盘旋转角速度 (rad/s) | 逆时针 |

**轮组编号 (俯视图):**

```
       Front (前)
   ┌───────────┐
   │ 0       3 │   0: 左前 LF
   │           │   1: 左后 LB
   │ 1       2 │   2: 右后 RB
   └───────────┘   3: 右前 RF
       Back  (后)
```

## 舵轮底盘
- **8 电机**: `motors[0..3]` 驱动 (3508) + `motors[4..7]` 转向 (6020)
### 运动学推导

舵轮的核心思路：**将底盘中心速度分解到每个轮组的局部坐标系，计算该轮组的线速度方向（即转向目标）和线速度大小（即驱动速度）。**

**底盘速度 → 各轮组局部速度**

以左前轮 LF 为例，其位置在 `(+wheel_r·√2/2, +wheel_r·√2/2)`:

```
local_vx_LF = vx + vw × (wheel_r × 0.707)
local_vy_LF = vy + vw × (wheel_r × 0.707)
```

其中 `vw × wheel_r × 0.707` 是底盘旋转在 LF 位置产生的切向线速度分量。

四个轮组的局部速度矩阵:

| 轮组 | `local_vx` | `local_vy` |
|------|-----------|-----------|
| LF (0) | `vx + vw·wheel_r·0.707` | `vy + vw·wheel_r·0.707` |
| LB (1) | `vx + vw·wheel_r·0.707` | `vy - vw·wheel_r·0.707` |
| RB (2) | `vx - vw·wheel_r·0.707` | `vy - vw·wheel_r·0.707` |
| RF (3) | `vx - vw·wheel_r·0.707` | `vy + vw·wheel_r·0.707` |

**局部速度 → 驱动速度**

```
target_speed = sqrt(local_vx² + local_vy²) / radius_wheel_m × decele_ratio
```

**局部速度 → 转向角度**

```
vector_rad     = atan2(local_vy, local_vx)
target_abs_rad = align_rad[i] + vector_rad
```

其中 `align_rad[i]` 是机械零点对齐角——当 `vector_rad = 0` (轮组正前方) 时，转向电机的编码器读数。

### 配置结构体

```c
typedef struct {
    float wheel_r;          // 投影点到中心距离 (m)
    float radius_wheel_m;   // 轮子半径 (m)
    float decele_ratio;     // 减速比 (3508: 19:1)
    float align_rad[4];     // 机械零点角度 [LF, LB, RB, RF]
} Chassis_Swerve_Config_s;
```

## 麦克纳姆轮

```
辊子方向 (俯视):

         Front
     ┌───────────┐
     │ ╲       ╱ │    LF(0): ╲ 辊子 +45°
     │   ╲   ╱   │    RF(3): ╱ 辊子 -45°
     │   ╱   ╲   │    LB(1): ╱ 辊子 -45°
     │ ╱       ╲ │    RB(2): ╲ 辊子 +45°
     └───────────┘
         Back
```

### 工作原理

麦克纳姆轮的辊子与地面接触时，轮子旋转产生的摩擦力方向与辊子轴向垂直。由于辊子倾斜 45°，轮子正向旋转时产生斜向 45° 的推力。

X型布置的关键特性:

| 轮组 | 辊子角 | 正转产生推力方向 |
|------|--------|-----------------|
| LF (0) | +45° | 前-左 (↖) |
| RF (3) | -45° | 前-右 (↗) |
| LB (1) | -45° | 后-右 (↘) |
| RB (2) | +45° | 后-左 (↙) |

通过组合四个轮子的转速，可以实现任意方向的平移 + 旋转。

### 逆运动学公式

```
ω_LF = (+vx - vy - ω·L) / R    [辊子 +45°]
ω_RF = (+vx + vy + ω·L) / R    [辊子 -45°]
ω_LB = (+vx + vy - ω·L) / R    [辊子 -45°]
ω_RB = (+vx - vy + ω·L) / R    [辊子 +45°]

其中:
  L = (wheel_base_x + wheel_base_y) / 2  等效旋转力臂
  R = wheel_radius                        轮子半径
  输出 = ω_i × decele_ratio              电机轴角速度 (rad/s)
```

### 运动举例

**纯前进 (vx > 0, vy = 0, ω = 0):**

```
ω_LF = ω_RF = ω_LB = ω_RB = vx / R  →  四轮同速正转 → 前进
```

**纯横移 (vx = 0, vy > 0, ω = 0, Y+ = 左移):**

```
ω_LF = -vy/R  (反转)    ω_RF = +vy/R  (正转)
ω_LB = +vy/R  (正转)    ω_RB = -vy/R  (反转)

→ LF/RB 反转, RF/LB 正转 → 辊子合力指向左 → 左移
```

**纯旋转 (vx = 0, vy = 0, ω > 0, CCW):**

```
ω_LF = -ωL/R  (反转)    ω_RF = +ωL/R  (正转)
ω_LB = -ωL/R  (反转)    ω_RB = +ωL/R  (正转)

→ 左侧两轮反转, 右侧两轮正转 → 逆时针旋转
```

### 3.5 配置结构体

```c
typedef struct {
    float wheel_base_x;   // 前后轮距 (m)
    float wheel_base_y;   // 左右轮距 (m)
    float wheel_radius;   // 轮子半径 (m), 通常 0.076 (76mm)
    float decele_ratio;   // 减速比 (3508: 19:1)

```

## 全向轮

**与麦轮的差异:**

| 特性 | 麦轮 (Mecanum) | 全向轮 (Omni) |
|------|---------------|---------------|
| 辊子方向 | 与轮平面成 45° | 与轮平面平行 (0°) |
| 施力方向 | 与辊子垂直，即与轮平面成 45° | 沿轮平面方向 |
| 轮平面朝向 | 全部朝前 (0°) | 指向外侧 45° |
| 侧向自由滚动 | 辊子斜 45° 实现 | 辊子平行轮平面实现 |

### 轮平面布局

```
         Front
     ┌───────────┐
     │ ↗       ↖ │    LF(0): ↗  轮平面 +45°  (施力: 前-左)
     │   ╲   ╱   │    RF(3): ↖  轮平面 -45°  (施力: 前-右)
     │   ╱   ╲   │    LB(1): ↖  轮平面 +135° (施力: 后-左)
     │ ↖       ↗ │    RB(2): ↗  轮平面 -135° (施力: 后-右)
     └───────────┘
         Back
```

全向轮只能沿其轮平面方向施力，垂直于轮平面的方向由辊子自由滚动。

### 逆运动学推导

对于位置 `(px, py)` 处、轮平面方向角为 `θ` 的全向轮:

```
ω_i × R = (vx - ω·py) × cos(θ) + (vy + ω·px) × sin(θ)
```

代入四个轮的位置和角度:

```
LF: px=+a, py=+b, θ= +45°  →  ω_LF = (+vx + vy + ω·(a-b)) / (R·√2)
RF: px=+a, py=-b, θ= -45°  →  ω_RF = (+vx - vy - ω·(a-b)) / (R·√2)
LB: px=-a, py=+b, θ=+135°  →  ω_LB = (-vx + vy - ω·(a-b)) / (R·√2)
RB: px=-a, py=-b, θ=-135°  →  ω_RB = (-vx - vy + ω·(a-b)) / (R·√2)

其中:
  a = wheel_base_x / 2   前后半轮距
  b = wheel_base_y / 2   左右半轮距
  R = wheel_radius       轮子半径
```

### 正方形底盘特性 (a == b)

当底盘为正方形 `(a == b)` 时，旋转差分项 `ω·(a-b) = 0`，旋转项从 ω_LF 和 ω_RB 中消失。此时纯旋转仅由 RF 和 LB 承担:

```
纯旋转 (CCW, ω > 0):
  ω_LF = 0   (自由滚动)
  ω_RF > 0   (正转驱动)
  ω_LB > 0   (正转驱动)
  ω_RB = 0   (自由滚动)
```

非正方形底盘 `(a ≠ b)` 时，四轮均参与旋转。
