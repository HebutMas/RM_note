## 1. 概述

### 1.1 本质：与串联轮腿是同一种机构

偏置并联轮腿和串联轮腿**本质上是同一种机构**——两个髋关节电机安装在同一水平线上，共同驱动一个平面五连杆并联机构，末端带动轮子。

```
两者相同的五连杆机构（俯视/侧视一致）：

        A ──────────── D          ← 两个电机在同一水平线上
        │＼            ／│
        │  ＼        ／  │        ← L₁, L₂, L₃, L₄ 四根连杆
        │    ＼    ／    │
        │      ＼／      │
        B ───── C ───── E
                        │
                       轮轴 (轮毂电机)

A、D = 两个 DM 髋关节电机（MIT 模式）
B、E = 被动关节
C   = 末端（轮轴）
```

**唯一的区别**：两个固定铰点 A 和 D 是否关于机体中线对称。

| 安装方式   | 对称安装（俗称"串腿"）    | 非对称安装（俗称"偏置并联"）              |
| ------ | --------------- | ---------------------------- |
| A、D 位置 | 关于机体中线**对称**    | 关于机体中线**偏置**（偏在一边）           |
| 质心特性   | 近似在对称面上，平衡时 θ≈0 | 质心随腿长变化偏移 → 需要 θ_comp(l) 补偿表 |
| 其余一切   | **完全相同**        | **完全相同**                     |

郑派成论文里的"偏置"是因为巡检机器人机身宽度约束，电机只能偏着装——**这是工程约束导致的，不是为了创新**。

### 1.2 本文档涵盖的内容

论文机器人虽然机构与串腿相同，但实现了一套更完整的控制系统。本文档记录其**通用技术贡献**（以下全部不依赖于偏置机构，可直接用于串腿）：

- LQR + 增量 MPC 级联控制（解决 Kalman 滤波器破坏 LQR 鲁棒性的问题）
- 龙伯格观测器优化二阶微分（替代差分+低通）
- PSO 自动优化 LQR 参数
- 大风车式翻倒自救（需腿能 360° 旋转）
- 撞击越障 + 跳跃过障 + 主动柔顺控制
- Kalman 打滑补偿 + 运动估计
- Simscape 仿真验证框架

## 2. 机械结构

### 2.1 整体架构

机器人由以下核心结构组成（来源：论文第二章）：

```
┌──────────────────────────────────────────┐
│              机体（碳纤维框架）             │
│  ┌─────────────────────────────────┐    │
│  │         二层：电控+电源           │    │
│  ├─────────────────────────────────┤    │
│  │         一层：动力+传感           │    │
│  └─────────────────────────────────┘    │
│      ↑ 髋关节固定件 ↑                    │
│     ┌┴────────────┴┐                    │
│     │  左腿   右腿   │                    │
│     │  (五连杆机构)  │  ← 两条腿对称      │
│     └──────┬───────┘                    │
│        轮毂电机 ×2                        │
└──────────────────────────────────────────┘
```

每条腿的髋关节**两个电机安装在同一水平线上**，驱动平面五连杆机构。论文中电机安装位置关于机体中线不对称（偏置），导致质心随腿长变化而偏移。

### 2.2 髋关节固定件

- 连接腿部与机体的关键承力件
- 承受巨大交变载荷
- 经 Inspire 拓扑优化（施加 800N 压力进行仿真验证）
- 结构无明显应力集中，强度满足要求

### 2.3 腿部轴系设计

- 为实现**360° 全周旋转**（大风车自救的核心基础），需要精细的轴系设计
- 包含：轴承座、传动轴、碳纤维大小腿板、气弹簧安装座
- 轴系剖面设计见论文图 2.7

### 2.4 碳纤维板材拓扑优化

- 大腿碳纤维板：固定一端，另一端施加 800N 压力 → 最大形变 **1.8mm**
- 小腿碳纤维板：同上条件 → 最大形变 **2.5mm**
- 结论：形变极小可忽略，结构强度满足要求

### 2.5 气弹簧

- 安装在五连杆机构中，提供被动弹力
- 作用：辅助起跳（跳跃越障）、落地缓冲
- 参数选择直接影响跳跃高度和收腿性能

## 3. 单腿动力学建模

### 3.1 模型简化

将偏置并联轮腿单腿简化为一个**变杆长的二阶倒立摆**（Second-Order Inverted Pendulum），原因：
- 单腿质量（1.47kg）远小于机体质量（17.36kg），不足 1/10 → 轻杆假设成立
- 五连杆机构的等效效果可用虚拟杆长 `l` 和摆动角 `θ` 描述
- 轮毂电机提供水平驱动力 `[公式]`，髋关节提供等效力矩 `[公式]`

### 3.2 变量约定

| 符号 | 含义 | 正方向 | 单位 |
|------|------|--------|------|
| `θ` | 摆杆与竖直方向的夹角 | 顺时针 | rad |
| `x` | 驱动轮位移 | 向右 | m |
| `φ` | 机体与水平方向的夹角 | 逆时针 | rad |
| `τ_w` | 轮向电机输出力矩 | 输出轴顺时针旋转 | N·m |
| `τ_h` | 髋关节电机等效输出力矩 | 输出轴顺时针旋转 | N·m |
| `F_x` | 驱动轮对摆杆的水平力 | 向右 | N |
| `F_y` | 驱动轮对摆杆的竖直力 | 向上 | N |
| `f` | 地面对驱动轮的摩擦力 | 向右 | N |

### 3.3 参数约定

| 符号 | 含义 | 单位 |
|------|------|------|
| `r` | 驱动轮半径 | m |
| `l₁` | 摆杆质心到驱动轮中心距离 | m |
| `l₂` | 摆杆质心到髋关节等效转轴距离 | m |
| `l₃` | 机体质心到髋关节等效转轴距离 | m |
| `m_w` | 驱动轮质量（不含轮向电机） | kg |
| `m_p` | 摆杆质量（含轮向电机，不含髋关节电机） | kg |
| `m_b` | 机体质量（含髋关节电机） | kg |
| `J_w` | 驱动轮转子转动惯量 | kg·m² |
| `J_p` | 摆杆绕质心转动惯量 | kg·m² |
| `J_b` | 机体绕质心转动惯量 | kg·m² |

### 3.4 动力学方程推导

#### 步骤 1：驱动轮方程

对驱动轮应用牛顿第二定律和转动定律：

```
水平方向: m_w · ẍ = f - F_x                    (3.1a)
竖直方向: 0 = N - F_y - m_w · g                (3.1b)
转动:     J_w · (ẍ/r) = τ_w - f · r            (3.1c)
```

消去摩擦力 `f`，得到驱动轮最终方程。

#### 步骤 2：摆杆和机体方程

由于摆杆和机体均处于**非惯性系**，采用质心运动定律：

- 在 x 轴和 y 轴分别应用质心系牛顿第二定律
- 应用质心系转动定律
- 共导出 **6 个耦合方程**

#### 步骤 3：完整动力学方程

综合驱动轮方程和摆杆/机体方程，得到**完整单腿非线性动力学方程**（形式如式 3.5）：

```
[公式] · [ẍ, θ̈, φ̈]ᵀ = [公式](x, θ, φ, ẋ, θ̇, φ̇, τ_w, τ_h)
```

该方程包含大量的三角函数和嵌套，是一个复杂的**非线性系统**。

> **模型选择说明**
>
> 本论文选择了二阶倒立摆（牛顿力学），而串腿文档选择了 WIP（拉格朗日力学）。这**不是机构差异导致的**——两种模型都可以描述同一套五连杆机构。只是二阶倒立摆将腿抽象为可变杆长 `l(t)`，更方便处理腿长时刻变化的场景（论文腿长范围 0.119→0.288m 较大）；WIP 模型则将腿长当参数固定后再线性化。选哪个模型是**建模偏好**，不是机构决定。

## 4. 状态空间方程与线性化

### 4.1 状态向量与控制向量

定义：
```
状态向量 x = [θ, θ̇, x, ẋ, φ, φ̇]ᵀ   (6维)
控制向量 u = [τ_w, τ_h]ᵀ             (2维)
```

非线性动力学记为 `ẋ = f(x, u)`。

### 4.2 雅可比线性化

在平衡位置 `x_eq = [0, 0, 0, 0, 0, 0]ᵀ` 附近进行雅可比线性化：

```
A = ∂f/∂x |_{x=x_eq, u=0}
B = ∂f/∂u |_{x=x_eq, u=0}
```

得到线性定常系统：

```
ẋ = A(l) · x + B(l) · u
```

**关键发现**：矩阵 A 和 B 依赖于腿长 `l`，因此不同腿长下需要不同的 LQR 增益矩阵。

### 4.3 变腿长处理

由于腿长 `l` 在实际运行中时刻变化：
- 在 MATLAB 中对不同 `l` 值离线求解 LQR
- 用**三次多项式**拟合反馈矩阵 `K(l)` 与 `l` 的关系
- 嵌入式运行时通过 `l` 实时查表/插值获得 `K`

拟合效果：曲线平滑穿过所有离散点，拟合度良好（见论文图 3.3）。

## 5. LQR 控制器设计

### 5.1 LQR 基本原理

对于线性定常系统 `ẋ = Ax + Bu`，构造代价函数：

```
J = ∫₀^∞ (xᵀQx + uᵀRu) dt
```

其中：
- **Q 矩阵**（对角阵）：惩罚状态偏差（状态权重越大 → 收敛越快 → 可能超调）
- **R 矩阵**（对角阵）：惩罚控制量（控制权重越大 → 越保守 → 响应越慢）

最优控制律为 `u = -Kx`，其中 `K = R⁻¹BᵀP`，`P` 为代数 Riccati 方程的解：

```
AᵀP + PA - PBR⁻¹BᵀP + Q = 0
```

### 5.2 带位移期望的控制律

为了实现速度/位移的人为控制，对控制律做调整：

```
u = -K(l) · [x - x_desired]
```

其中 `x_desired = [θ_offset, 0, x_desired, 0, 0, 0]ᵀ`

### 5.3 质心偏移补偿（偏置安装的额外处理）

**这是偏置安装与对称安装唯一的功能差异**。

由于电机安装位置不关于机体中线对称，不同腿长时等效质心位置不同，导致平衡摆角不是固定的 θ=0，而是随腿长变化：

> 串联五连杆（对称安装）的质心近似在五连杆对称面上，平衡时摆角 θ=0 即可。偏置安装的质心会随腿长变化而偏移，若不加补偿，控制器会自行"找"到一个奇怪的平衡角度，表现为机器人站立时总是歪向一侧。

**补偿方法**：
- 对不同的腿长 `l`，实物测试摆杆角度补偿值 `θ_comp(l)`
- 同样使用三次多项式拟合 `θ_comp(l)` vs `l`
- 运行时：`θ_desired = θ_comp(l)` → 控制器自动将平衡位置偏移到质心正上方

实测补偿数据（论文表 3.3）：

| 腿长 l (m) | 0.119 | 0.155 | 0.243 | 0.288 |
|-----------|-------|-------|-------|-------|
| θ_comp (rad) | 0.075 | 0.036 | 0.000 | -0.004 |

## 6. LQR + 增量MPC 级联控制

### 6.1 为什么需要 MPC

LQR 理论上有良好的鲁棒性裕量（增益裕度 > 6dB，相位裕度 > 60°），但**引入 Kalman 滤波器后这些裕量会失效**。而状态观测器（如 Kalman 滤波器）在实际系统中**不可避免**。

为了解决这个问题，在 LQR 控制器后**串联一个增量 MPC 控制器**，提供额外的鲁棒性补偿。

### 6.2 增量 MPC 的核心思想

传统 MPC 需要在每个控制周期求解二次规划（QP）问题 → **嵌入式 MCU 无法实时计算**。

增量 MPC 的巧妙之处：
1. 构造**增量式状态空间模型**（差分状态和输入）
2. 将 QP 问题转化为**直接求导求极值**
3. 得到一个**解析解** `ΔU = [公式]`，无需在线优化

**代价**：预测补偿效果有所折扣，但仍能提供足够的鲁棒性。

### 6.3 算法流程

```
步骤 1：离散化状态空间方程
    x(k+1) = A_d · x(k) + B_d · u(k)

步骤 2：构造增量模型（取 Δx(k) = x(k) - x(k-1)，Δu(k) = u(k) - u(k-1)）
    [Δx(k+1); y(k+1)] = [A_d 0; CA_d I] · [Δx(k); y(k)] + [B_d; CB_d] · Δu(k)

步骤 3：取预测域 N_p，构建预测矩阵 F、Φ
    未来 N_p 步的状态 = F · 当前增广状态 + Φ · 未来 N_p 步的控制增量

步骤 4：构造代价函数（状态约束 + 控制增量约束）
    J = (F·z + Φ·ΔU)ᵀQ_bar (F·z + Φ·ΔU) + ΔUᵀR_bar ΔU

步骤 5：对 ΔU 求导求极值
    ΔU* = -(ΦᵀQ_bar Φ + R_bar)⁻¹ · ΦᵀQ_bar · F · z
```

### 6.4 实际部署策略

在实物机器人中：
- MPC 补偿矩阵的第二行（对应髋关节力矩补偿）被**反向叠加**到 LQR 的髋关节力矩输出
- 效果：**减小轮向电机的期望力矩**，避免轮电机物理极限导致的模型偏差
- 补偿矩阵的系数同样依赖腿长 `l`，使用**双指数函数**拟合（而非三次多项式）

### 6.5 级联架构总结

```
期望状态 x_des ──→ [LQR 控制器] ──→ u_LQR ──→ (+) ──→ [五连杆动力学变换] ──→ 关节扭矩
                          ↑                        ↑
                          |                        |
                  [增量 MPC 补偿器] ──→ Δu_MPC ──→ (+)
                          ↑
                   当前状态 x
```

> **设计哲学**：LQR 负责"最优"，MPC 负责"鲁棒"。LQR 提供 baseline 控制，MPC 弥补 Kalman 滤波器引入后的鲁棒性损失。这种设计是专为嵌入式 MCU 计算的工程化选择——在线求解 QP 是不可能的，但求导求解析解是可以的。

## 7. 五连杆运动学与动力学变换

> 📘 这部分是标准五连杆并联机构的运动学/动力学变换，**与电机安装是否对称无关**——对称安装的串腿同样适用。

### 7.1 变换的必要性

控制器输出的是"虚拟量"：
- `τ`：髋关节电机对旋转中心的合成等效力矩
- `F_l`：沿腿长方向的推力

但实际执行器需要的是"物理量"：
- `q₁, q₂`：两个髋关节电机的目标角度
- `τ₁, τ₂`：两个髋关节电机的目标力矩

运动学与动力学变换就是这两层之间的**桥梁**。

### 7.2 轻杆假设

由于腿部质量（~1.47kg）远小于机体质量（~17.36kg），将五连杆视为**轻杆模型**。这大大简化了动力学变换——可以直接使用**虚功原理（VMC 思路）**进行力映射。论文中两个电机安装位置不对称，但这不影响轻杆假设和 VMC 方法——对称安装的串腿同样适用。

### 7.3 正运动学变换

**已知**：`q₁, q₂`（两个髋关节电机角度）
**求解**：`θ, l, α, β`（摆杆角度、腿长、气弹簧相关变量）

核心几何关系（论文式 3.34-3.44）：
```
C_x = A_x + L₁·cos(q₁) + L₂·cos(q_passive1)
C_y = A_y + L₁·sin(q₁) + L₂·sin(q_passive1)
    = D_x + L₄·cos(q₂) + L₃·cos(q_passive2)
    = D_y + L₄·sin(q₂) + L₃·sin(q_passive2)
```

通过闭环约束求解被动关节角，进一步得到：
```
θ = atan2(C_x - O_x, C_y - O_y)   —— 摆杆角度
l = √[(C_x - O_x)² + (C_y - O_y)²]   —— 等效腿长
```

同时求导得到速度级雅可比：
```
[θ̇; l̇]ᵀ = J_K · [q̇₁; q̇₂]ᵀ
```

### 7.4 逆运动学变换

**已知**：`θ, l`（期望摆杆角度和腿长）
**求解**：`q₁, q₂`（两个髋关节电机目标角度）

- 先由 `θ, l` 得到 C 点坐标
- 由几何约束反解 `q₁, q₂`（论文式 3.45-3.48）

### 7.5 正动力学变换（VMC 虚功映射）

基于虚功原理：

```
τ₁·δq₁ + τ₂·δq₂ = τ·δθ + F_l·δl
```

即：
```
[τ₁; τ₂] = J_Dᵀ · [τ; F_l]
```

其中 `J_D` 为动力学雅可比矩阵，由运动学雅可比 `J_K` 推导而来。

**实际使用中还需叠加**：
- 气弹簧力矩补偿
- 重力补偿

### 7.6 变换链路总结

```
控制器输出 (τ, F_l)
     │
     ▼
[正动力学变换]  J_Dᵀ · [τ; F_l] + 气弹簧补偿 + 重力补偿
     │
     ▼
髋关节力矩 (τ₁, τ₂)  ──→  电机电流指令

期望腿姿态 (θ, l)
     │
     ▼
[逆运动学]  反解 (q₁, q₂)
     │
     ▼
髋关节角度 (q₁, q₂)  ──→  电机位置指令
```

## 8. 双腿耦合与功能算法

### 8.1 腿长控制

两种模式：

**模式 A：绝对腿长控制**
- 采用前馈 + PD 控制
- 前馈：抵抗机体重量 → `F_ff = mg · cos(θ)`
- PD：形成虚拟弹簧-阻尼效应 → 减震效果
- 实测腿长变化极小

**模式 B：离地高度控制**
- 用于越障和特殊工况
- 根据障碍高度和当前腿长动态计算目标腿长

### 8.2 翻滚角补偿

**问题**：地面不平 → 左右腿着地高度不同 → 机体侧倾

**方法 1：腿长补偿**（基于几何计算）
```
tan(α_ground) = Δl_legs / d_base   —— 地面坡度
Δl_compensation = d_base · sin(α_ground) / 2   —— 半差分法计算腿长差
```

**方法 2：支撑力补偿**（基于 LQR）
- 构建翻滚轴状态空间方程
- 独立计算翻滚补偿力矩
- 该方法允许离地高度动态变化

### 8.3 双腿协调（防劈叉）

两条腿各自运行独立的 LQR+MPC 控制器，但接收**相同的速度指令**和**通过翻滚补偿分配的独立腿长指令**。

防劈叉 PD 控制器：
```
τ_split = K_p_split · (θ_left - θ_right) + K_d_split · (θ̇_left - θ̇_right)
```

### 8.4 转向控制

**基本方法**：左右轮差速
```
v_left = v_desired - ω_desired · d_base/2
v_right = v_desired + ω_desired · d_base/2
```

**Lyapunov 预瞄控制**：
- 预测转弯时机体的倾斜趋势
- 基于 Lyapunov 方法推导预瞄时间
- 提前补偿横滚力矩

**转弯时双轮力矩分配**：
```
τ_roll = [通过 LQR 额外计算]
τ_left = τ_forward + τ_roll/2
τ_right = τ_forward - τ_roll/2
```

### 8.5 离地检测

**原理**：通过估计支撑力（驱动轮法向力）判断是否离地。

**支撑力估计**：
```
F_support = [通过机体竖直加速度 + 轮加速度 + 力平衡推导]
```

当 `F_support < threshold` → 判定为离地。

**空中姿态控制**：
- 清空 LQR 状态（除摆杆角度外）
- 轮毂力矩归零
- 保持腿竖直 → 准备落地

### 8.6 大风车式翻倒自救

这是偏置并联轮腿的**关键创新功能**——利用腿部 360° 全周旋转能力实现翻倒恢复。

#### Step 1：腿部旋转速度环

通过重力补偿 + PD 控制实现匀速旋转：

```
τ_gravity_comp = m_leg · g · d_cm · cos(θ_leg + φ_offset)
τ_rotation = K_p · (ω_desired - ω_actual) + τ_gravity_comp
```

实物中以 `[公式]` 角速度恒速转动，得到平稳的角速度曲线。

#### Step 2：腿部姿态纠正

判断条件：当 `θ_leg` 处于 `[公式]` 到 `[公式]` 或 `[公式]` 到 `[公式]` 区间时，腿部向后摆到同步位置。

#### Step 3：机体翻转处理

**情况 1（向后翻转）**：
1. 两条腿同步旋转到指定位置
2. 逆时针同步旋转 → 用腿"别"地 → 机体恢复水平

**情况 2（向前翻转）**：
1. 两条腿同步旋转到指定位置
2. 顺时针同步旋转 → 别车 → 机体恢复水平
3. 然后自动进入腿部姿态纠正

#### Step 4：起身

1. 控制 `θ_leg` 闭环到同步位置
2. 接着闭环到平衡位置附近的 `θ` 值
3. 最后启动 LQR 控制器 → **丝滑起身**

### 8.7 失控检测与失效处理

检测条件：
- `θ` 偏离超过阈值
- `φ` 偏离超过阈值
- 翻滚角偏离超过阈值

处理：
- 禁用全部电机
- 复位数据
- 重新初始化系统

### 8.8 撞击式跨越台阶型障碍

**核心策略**：碰撞 → 收腿 → 自起

#### 碰撞力估计

对驱动轮进行受力分析：
```
F_collision = [通过机体水平加速度 + 轮加速度 + 力平衡推导]
```

检测：两轮碰撞力平均值 > 阈值 → 碰撞发生。

#### 越障动作序列

1. **伸高腿长** → 机体趴在障碍高处
2. **碰撞检测** → 两轮碰撞力均值超过阈值
3. **停止 LQR** → 停止平衡控制
4. **收腿** → 将腿收到障碍高处
5. **重新自起** → 恢复 LQR 控制

**实测结果**：稳定快速跨越 **200mm** 台阶障碍，通过率**接近 100%**。

### 8.9 跳跃过障

**适用场景**：障碍高度超过 `[公式]`（碰撞越障的极限）

#### 跳跃模型（基于机械能守恒）

**起跳阶段**：
```
½·m·v_y² = ∫_{l_min}^{l_max} (F_thrust - m·g) · dl
v_y_liftoff = √[2/m · ∫(...)dl]
```

**空中阶段（斜抛运动）**：
```
H_max = v_y_liftoff² / (2g)     —— 最大高度
t_peak = v_y_liftoff / g          —— 达到最高点时间
```

**收腿阶段**：
```
F_retract = [通过推力+收腿长度+时间约束求解]
```

**落地阶段**：
```
t_fall = √[2(H_max - h_obstacle) / g]
```

#### 跳跃过程中的姿态优化

发现问题：空中保持 `θ` 竖直 → 产生转矩 → 机体翻转

解决方案：空中**保持机体水平**而非摆杆竖直 + 落地前主动伸腿缓冲

**实测结果**：
- 含气弹簧配置：最高 **420mm** 跳跃
- 优化后稳定跳跃：**200mm**（论文图 3.29）
- 存在限制：电机无法同时满足高转速 + 高转矩 → 跳跃高度和收腿长度一般低于计算值

### 8.10 主动柔顺控制

**问题**：位移控制模式下撞墙 → 位移误差持续累加 → 系统发散

**解决思路**：估计机体所受推力 → 调整位移期望 → 顺应外力

**推力估计**（简化模型）：
```
F_external = [基于轮向力矩 + 机体加速度的力平衡推导]
```

实际使用中由于轮电机减速箱力矩噪声大，将 `τ_w` 项置零，误差可接受。

**柔顺控制器流程**：

```
外力估计 F_ext ──→ [柔顺律] ──→ Δx_desired
                                      ↓
期望速度 v_des ──→ (+) ──→ [LQR+MPC] ──→ 电机
                    ↑
              -Δx_desired (顺应外力方向)
```

本质是：当感知到外部推力时，主动让位移期望"退让"，形成**虚拟阻尼/弹簧**效果。

## 9. 观测器设计

### 9.1 运动估计与打滑补偿

**问题**：不能直接取左右轮向电机角速度平均作为机体速度，原因有三：
1. 需要考虑**定子侧的转动和平动**
2. 轮子会**打滑**
3. 几何关系需正确转换

**速度解算**：
```
ω_wheel_rotor_to_ground = ω_rotor_to_stator + ω_stator_to_body + ω_body_to_ground
v_wheel = ω_wheel_rotor_to_ground · r     —— 轮向水平速度
```

同时从几何关系求解：
```
v_body_from_geometry = [由 θ, l, θ̇, l̇ 推导]
```

最终速度 = 算术平均（轮速法 + 几何法）

**打滑补偿卡尔曼滤波器**：

状态向量：`[v_x, bias_accel]ᵀ`（速度和加速度计偏差）
测量向量：`[v_encoder, a_accel]ᵀ`（编码器速度和加速度计数据）

卡尔曼滤波融合：
- 编码器速度 → 有打滑但有绝对参考
- 加速度计 → **不会打滑**但有漂移

→ 卡尔曼滤波器自然滤除打滑引起的编码器速度噪声

### 9.2 龙伯格（Luenberger）观测器优化二阶微分

**问题**：支撑力估计和碰撞力估计都需要 `θ̈` 和 `l̈`（摆杆角度和腿长的二阶导数）。直接差分 + 低通滤波 → **大量尖峰噪声**。

**Luenberger 观测器方案**：

假定二阶微分为 0，构建离散观测器：

```
预测步:  x̂⁻(k+1) = A · x̂(k) + B · u(k)
更新步:  x̂(k+1) = x̂⁻(k+1) + L · (y(k) - C · x̂⁻(k+1))
```

对 `θ` 和 `l` 分别构造状态向量：
```
x_θ = [θ, θ̇, θ̈]ᵀ
x_l = [l, l̇, l̈]ᵀ
```

**实测效果**（论文图 4.2）：
- 红线（差分+低通滤波）：大量尖峰噪声
- 绿线（Luenberger 观测器）：**曲线平滑很多**，尖峰噪声显著减少

## 10. PSO 参数优化

### 10.1 问题

LQR 的 Q 和 R 矩阵通常靠手动调试，这是**不科学的**：
- 容易参数不佳 → 鲁棒性差
- 调试效率低
- 多目标难以权衡

### 10.2 PSO 优化方案

**优化目标**：仅优化 R 矩阵（控制惩罚矩阵），Q 保持固定

**适应度函数**（8:2 加权）：
```
Fitness = 0.8 × (轮向力矩峰值) + 0.2 × (机体俯仰角峰值)
```

- 轮向力矩尽量小 → 更高鲁棒性（80% 权重）
- 俯仰角变化尽量小 → 更高平稳性（20% 权重）

**评估平台**：Simscape 仿真模型（参数与实物一致），运行全流程 + 越障过程

**PSO 参数**：30 个粒子，100 次迭代

### 10.3 优化结果

| 参数 | 原 R 矩阵 | PSO 优化后 |
|------|----------|-----------|
| R 对角元 | diag(10, 10, 10, 10) | 约 diag(3.85, 5.97, 11.03, 7.28) |
| 轮向峰值力矩 | ~2.25 N·m | ~1.45 N·m |
| 力矩尖峰次数 | 较多 | **显著减少** |

**结论**：PSO 优化有效降低了峰值力矩和尖峰频率，达到优化目的。虽不能完全替代手动调参，但**减少了调参数量，改善了调参效果**。
